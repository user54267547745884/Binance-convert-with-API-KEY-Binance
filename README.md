# Binance-convert-with-API-KEY-Binance
Este repositorio es una reconstrucción de código generado por IA tomado de <a href="https://www.perplexity.ai">Perplexity.ai</a> y Google IA, que implementa el proceso Binance Convert SWAP en la red mainnet utilizando una clave API.


<h2>Instalación de entorno termux/debian/ubuntu</h2>

```bash
apt-get install nodejs -y
```

```bash
npm install crypto axios
```

<h2>Ejecución de orden en Binance Convert via shell en Termux/Debian/Ubuntu</h2>

```bash
node binance-convert-convertion.js
```


<br>

<h2>Como obtener API KEY y API SECRET</h2>


<p>El API SECRET solamente aparece en la creación de API KEY una vez posibilitada para copiar y obtener acceso para la utilización de API SECRET, por esta razón es necesario copiar y almacenar en el momento de obtener acceso a la API SECRET la API SECRET.</p>
